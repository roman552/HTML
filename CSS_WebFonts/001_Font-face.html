В предыдущих версиях css разработчики были вынуждены использовать только те шрифты, которые гарантированны установлены на компьютере пользователя в css3 разработчики могут использовать любые шрифты которые они захотят. Когда вы найдете необходимый шрифт просто разместите его на веб-сервере и подключите его с помощью нового css3 свойства @font-face. Подключенный шрифт будет скачан и отображет автоматически при загрузке страницы.

обратите внимание: internet explorer поддерживает шрифты в формате .eot а остальные браузеры в формате .ttf т.е. для того чтобы шрифт отображался корректно во всех браузерах вы должны подключить его к странице в двух форматах
